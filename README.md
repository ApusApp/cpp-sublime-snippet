## C++ Sublime Text Snippets

Made for C++11.

Uses this style:

```cpp
if(/* condition */) {
    /* code */
}
```

## Installation

At the moment the only way to install this is to run `git clone` where you keep your packages. Maybe one day I'll add this to the Package Control.
